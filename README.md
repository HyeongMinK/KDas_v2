# KDasHboard

> 한국 산업연관표 기반 경제 네트워크 분석 대시보드

![Python](https://img.shields.io/badge/Python-3.10%2B-blue) ![Streamlit](https://img.shields.io/badge/Streamlit-1.20%2B-red)

---

## 개요

**KDasHboard**는 한국 은행 산업연관표(Input-Output Table)를 업로드하여 레온티에프(Leontief) 모형 기반의 경제 분석과 네트워크 분석을 수행하는 Streamlit 대시보드입니다.

이 도구의 핵심 기능은 **사용자가 직접 새로운 산업군을 정의하고 기존 산업을 재편한 뒤, 해당 신규 산업의 경제적 파급력(전방·후방 연관효과)을 정량적으로 분석**하는 것입니다. 예를 들어 기존 산업연관표에 존재하지 않는 신흥 산업(AI, TMT, M&E 등)을 기존 섹터에서 분리·추가하여 그 파급 효과를 측정할 수 있습니다.

---

## 주요 기능

### 신규 산업군 정의 및 파급력 분석
- 기존 산업 섹터에서 값을 이전(transfer)하거나 분할(divide)하여 **새로운 산업 섹터를 생성**
- 재편된 산업연관표를 바탕으로 전방연관효과(FL)·후방연관효과(BL) 분석
- 배치 처리(Excel/ZIP)를 통한 자동화된 산업 재편 작업

### 데이터 업로드 및 파싱
- 한국 산업연관표 Excel 파일 업로드
- 지원 포맷: 한국(2010–2020), 한국 구버전(1990–2005)
- 문자열 값 자동 감지 및 정제

### 데이터 편집
- **배치 처리**: Excel/ZIP 파일로 alpha 값(이전 비율) 기반 자동 편집
- **수동 편집**: 섹터 추가, 값 이전, 값 분할, 영·음수 제거 등 인터랙티브 조작
- 편집 이력(edit_ops) 추적으로 재현성 보장
- 기준·로컬 두 버전 동시 관리 및 비교

### 레온티에프 분석
- 입력계수행렬(A) 계산
- 레온티에프 역행렬 근사 산출 (무한급수 전개: I + A + A² + …)
- 전방·후방 연관효과(FL-BL) 시각화

### 네트워크 추출
- **방법 1**: 임계값 최적화 (거리 최소화 + 연결성 검증)
- **방법 2**: 무한급수 전개 (정보 누적이 수렴할 때까지 반복)
- 이진 네트워크 및 무방향 네트워크 변환

### 네트워크 중심성 지표 계산
| 지표 | 설명 |
|------|------|
| Degree Centrality | 진입/진출 연결 수 기반 중심성 |
| Betweenness Centrality | 최단 경로 상 위치 기반 중심성 |
| Closeness Centrality | 다른 노드까지의 평균 거리 기반 중심성 |
| Eigenvector Centrality | 연결된 노드의 중요도 반영 중심성 |
| HITS (Hub & Authority) | 허브·권위 점수 |
| Constraint & Efficiency | 구조적 공백(Structural Hole) 기반 지표 (Kim, 2021) |

### 결과 다운로드
- 원본 데이터, 편집 데이터, 행렬, 레온티에프 산출물, 네트워크 지표 등을 ZIP으로 일괄 다운로드

```bash

streamlit run app.py
```

---

## 사용법

```
1. Excel 파일 업로드
        ↓
2. 데이터 편집 (선택)
   - 배치 처리 또는 수동 편집으로 신규 산업 섹터 추가·재편
        ↓
3. 레온티에프 분석
   - 입력계수행렬 및 역행렬 계산
   - FL-BL 파급력 분석
        ↓
4. 네트워크 추출
   - 방법 1 (임계값 최적화) 또는 방법 2 (급수 전개) 선택
        ↓
5. 중심성 지표 산출
        ↓
6. 결과 ZIP 다운로드
```
